## stuff i've built / run

**CREST** *(private)* — internal account request and approval system.  
laravel + mysql + docker + nginx + microsoft 365 login. includes role-based workflow, approval matrix, audit logs, pdf generation, and email notifications.

**[phantix](https://github.com/imkaiwhyask/phantix)** *(wip)* — internal messaging desktop app. electron + php + mysql + docker. built it because we needed it.

**[helpit](https://github.com/imkaiwhyask/HelpIT)** *(wip)* — internal helpdesk, forked from [faveo](https://github.com/faveosuite/faveo-helpdesk). self-hosted, dockerized, sitting behind nginx.

**wazuh + grafana siem** — security monitoring pipeline. deployed, tuned, dashboards wired up. it’s live.

**m365 ↔ ad sso** — hybrid identity using microsoft entra connect. on-prem domain controllers syncing clean with microsoft 365.

**dc01 / dc02** — managing domain controllers, group policy, dns, users, and the whole directory. not just “active directory” on a resume, actually running it.

**vsphere + fortigate** — VMs, hypervisor management, VLANs, firewall rules, network segmentation, and troubleshooting when the network decides to be dramatic.
